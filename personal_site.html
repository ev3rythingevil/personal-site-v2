Project: Band Together

Band Together is an application created to help musicians find other local players in order to form a band. Upon signing up, members can click through profiles that appear on their main page, and chat with mutual matches. The app also uses location information to only show you users within a certain radius. I used self referential tables in Ruby on Rails to allow the users to match, and the front end is built on React.

Project: Tarot Reader

In this application, random tarot cards were generated for users, and displayed with a brief description of the card and it's meaning. Users could save cards to a database, so they could look back at the history of their readings. This app was built in React with a JSON server running as a mock back end.

Project: Monster Manual

The focus of this application was to make an interactive list of Monsters to be used in Dungeons and Dragons Fifth Edition games. A search bar allows users to find monsters by name, and once clicked, the Monster's information is displayed for the DM to use. Health is also subtractable/addable, allowing for dynamic usage during gameplay.
